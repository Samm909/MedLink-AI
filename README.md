# MedLink AI

## Overview

MedLink AI is an AI-powered medicine availability platform that helps users locate medicines from nearby pharmacies, discover generic alternatives, and receive AI-assisted medicine information.

## Problem Statement

People often visit multiple pharmacies to find a required medicine. MedLink AI aims to simplify this process by providing a centralized search platform with real-time pharmacy inventory.

## Features

### User Features

* Search medicines
* View nearby pharmacies
* Check medicine availability
* Find generic alternatives
* AI-powered medicine assistant
* Upload prescriptions (future enhancement)

### Pharmacy Features

* Manage inventory
* Update medicine stock
* View analytics
* Manage medicine catalog
